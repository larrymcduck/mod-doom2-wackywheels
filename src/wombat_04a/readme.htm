Wombat Game Tools 0.4a

Wombat basically lets you view the data in a some games from the early 90's, from classic gaming companies such as Apogee and Epic Megagames, and a few newer games to boot. It also lets you change the data in a few games, but this feature is new and not available for all games.

If you wanna talk to me - I'm at szevvy@hotmail.com
The official Wombat website is at http://student.ci.qut.edu.au/~n4738667

All you need to know:

  • To scroll images or levels, just use the mouse to click-and-drag.
  • The 'replace' button will NOT convert files to the format required by the game; files must be in the format that the game expects - use the 'import' button if you want to change images from bmp format to the format used by the game.

Wombat 0.4:

New features:

  • Games are now split into A-M and N-Z
  • 'Unknown image' now handles both EGA and VGA, plus 16-bit images

New games supported:

  • Keen 4-6, Keen Dreams : all tiles, sprites, and images (let me know if any versions of these games have problems)

  • Civilization 3: extract from any .zfs, view images and play sounds (no tiles or sprites yet)
  • Dangerous Dave: all images, tiles, and sprites - colours are incorrect, though

Games with increased support:

  • None in this release

History

Wombat 0.3:

New features:

  • Import bitmaps into:
    • Major Stryker: all expect fonts
    • Cosmo: all except story images and sprites, tile attributes cannot be set yet
    • Duke Nukum 1: fullscreen images
    • Captain Comic 1: fullscreen images
    • Duke 2: backdrops
    • Crystal Caves: fonts and statusbar
  • Bugfixes for multiple-file extraction, amongst others
  • More .voc support, 2 and 2.6 bit voc's now play

New games supported:

  • Halloween Harry, Alien Carnage: all tiles, sprites, and most other images
  • Baron Baldric: extract, view tiles
  • Thief 3: extract only

Games with increased support:

  • Hocus Pocus: Fixed up some names and things
  • Cannon Fodder 1 & 2: Drawing more images, colours are closer to what they should be
  • Simcity 2000: Drawing some images now, although all buildings and so on are still compressed
Wombat 0.2a:

New features:

  • Got rid of the keyboard scroll in level viewers and that nasty image scroller in images, and replaced it with mouse-drag to scroll everything.

New games supported:

  • Duke Nukem 3D (extract, play sounds)
  • Sid Meier's Pirates! (2004) (extract, view 5% of images (the rest are .dds))

Games with increased support:

  • Duke Nukem 2: fixed palette issues with a few sprite images
  • Hocus Pocus: Added support for version 1.1
Wombat 0.2:

New features:

  • Extract multiple files at a time - extract whole packed files if you wish, or just the images - whatever you need!
  • Wombat is now much faster and more stable - Duke 2 sprites are now fast!
  • Replace same-sized files in packed files all games except Monster Bash and Cannon Fodder 1 & 2
  • Replace any files in packed files for Cosmo, Major Stryker, Duke 2 and Wacky Wheels
  • Export .snd files to wav
  • Play and export most .voc files
  • Many improvements to unknown EGA/VGA image driver
  • Palette files are now seperated, and they draw
  • .exe string viewer tool has been added
  • Many improvements to program layout

New games supported:

  • Commander Keen 1-3: all graphics - thanks Andy Durdin!
  • Jill of the Jungle: all graphics
  • Xargon: all graphics (I hope)
  • Traffic Department 2192: all images
  • Dark Ages: most images
  • 7 cities of gold VGA: all .m52 images, about half have wrong colours
  • Jazz Jackrabbit: 'planet' images only
  • Prehistorik 1: extract all files, play sounds
  • Simcity 2000: extract all data, play sounds
  • Cannon Fodder 1 & 2 - extract all files except sounds, view fullsceen images and tiles

Games with increased support:

  • Wacky wheels: level viewer added (not complete)
  • Monster bash / ScubaVenture: decompression fixed
  • Gateworld: sprite and tile support added, that should be all graphics
  • Dune 2: fullscreen images and small images draw, no units or tiles yet
  • Major Stryker: font files show up now
  • Hocus Pocus: added file names for images and music, separating what I suspect is level tile-data into seperate folder
  • Clyde's Adventure: drawing all sprites now
  • Duke Nukem 2: level viewer added
  • Various small improvements to many games
